Design Philosophy

  1. no tutorial:

    Discovering and learning the game should be part of the fun.

  2. no single story:

    A gamestory is the story of a playthrough by a single player. Each player experiences an own story. It comes from inside the gameworld. Each game is like a machine to create stories but it is not a single story.

  3. no Levels:

    Create open worlds within the possibility spaces of player actions, not levels. The aplicability of actions should only be constrained by logic, not by levels. That is what makes games fun and players think. I like games that make players think. In my opinion this is the strength of the medium games:

    Player autonomie.
